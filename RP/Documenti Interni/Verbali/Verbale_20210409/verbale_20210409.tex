\section{Argomenti della riunione}
\begin{itemize}
    \item Scelta del servizio di Continuous Integration: il gruppo ha discusso sul miglior servizio da poter utilizzare per la Continuous Integration, scegliendo tra Github Actions, TravisCI, CircleCI e Vercel. Per il backend si è scelto di utilizzare TravisCI, siccome è già stato studiato da alcuni membri, mentre per il frontend si è scelto Vercel, che è un servizio offerto appositamente per piattaforme che utilizzano Next.js.
    \item Discussione sul workflow: per la scrittura del codice si è scelto di utilizzare Github flow.
    \item Discussione sugli incarichi per il weekend: il gruppo, per dividersi il carico di lavoro, ha deciso di suddividere le tecnologie da studiare tra i vari membri, così ognuno sia più specializzato in una o più determinate tecnologie.
\end{itemize}
